Guidance for use:
- Use `apply_master_ob(APPLY_<NAME>, nargs)` or `apply(APPLY_<NAME>, obj, nargs, ORIGIN_DRIVER)`.
- `apply_master_ob` returns (svalue_t *)-1 if the master object isn't loaded yet.
- The `edit_source` script generates `applies_table.c` and `applies.h` from the `applies` file.
- When asking the LLM to implement or modify master hooks, include the specific APPLY_<NAME>
  and expected argument/result contract (e.g., `APPLY_PRELOAD` should return objects to preload).
